Cambio de estilos de elementos con eventos de JQuery

Construir un HTML con dos input que implementen los eventos de “tomar foco” y “perder foco” cuya funcionalidad sería cambiar el estilo del texto de dichos elementos a un color rojo.

Si pierde el foco el texto el estilo del texto será rojo
